# C语言多功能通讯录管理系统（C语言课程大作业）
## 项目简介
本项目使用纯C语言开发Windows控制台通讯录管理系统，采用**二进制文件持久化存储**，无需数据库，划分多套独立结构体存储用户、联系人、短信、操作日志、VCF导入数据；集成账号登录注册、联系人CRUD、短信收发、分组管理、黑名单、纪念日提醒、VCF批量导入、脱敏导出、数据可视化统计、XOR加密备份恢复、自动清理过期临时联系人、全流程操作日志等拓展功能，功能完整，适合作为C语言期末综合大作业。

程序仅适配Windows系统，调用windows.h解决控制台中文乱码，所有数据本地持久化保存，重启程序自动加载历史数据。

## 项目文件说明
### 程序自动生成存储文件
1. `contact.bin`：主通讯录联系人核心二进制数据
2. `user.bin`：用户账号、密码存储文件
3. `log.bin`：系统全部操作日志记录
4. `sms.bin`：收发短信数据存储
5. `contact_main.bin`：VCF导入工具专用数据源
6. `backup_时间戳.bin`：XOR加密备份文件

### 导出生成文件
- `脱敏通讯录导出.csv`：CSV格式脱敏通讯录（手机号中间4位打码）
- `脱敏通讯录导出.txt`：文本格式化脱敏通讯录

## 完整系统功能清单
### 1. 账号登录注册模块
1. 新用户注册，账号密码持久化存入user.bin
2. 用户登录校验，无账号自动跳转注册页面
3. 登录成功自动执行：过期临时联系人清理、纪念日提醒，再进入主菜单

### 2. 联系人基础增删改查
1. 添加联系人：填写姓名、11位手机号、分组、分组ID、隐私等级、临时有效期、纪念日、备注；自动查重、自动生成运营商标签
2. 全量查看联系人：表格打印ID、姓名、手机号、分组、隐私等级、黑名单、标签、纪念日
3. 手机号精准检索联系人，展示联系人完整全部信息
4. 修改联系人姓名/分组/隐私等级/备注等信息，自动记录操作日志
5. 删除联系人，二次确认弹窗，删除同步写入日志
6. 联系人按姓名冒泡升序排序，排序操作留存日志

### 3. 联系人高级管理功能
1. 拉黑联系人标记，拉黑后无法接收短信，重复拉黑拦截校验
2. 根据分组名称筛选联系人
3. 根据分组ID筛选联系人
4. 自动清理过期临时联系人：程序登录自动判断过期数据，同步删除关联短信
5. 纪念日提醒：开机展示今日纪念日联系人；自动提示90天未联系人员
6. 自动生成标签：识别移动/联通/电信运营商；根据备注追加家人/同事标签；标记临时联系人、黑名单

### 4. 短信一体化管理
1. 单条发送短信：校验黑名单，陌生号码二次确认发送
2. 回复已接收短信，自动写入短信文件
3. 查看所有收到的短信记录
4. 模拟外部接收短信（测试工具）
5. 按分组ID批量群发短信，过滤黑名单联系人，操作记录日志

### 5. VCF导入 & 隐私脱敏导出工具
1. VCF手机通讯录批量导入：解析标准vcf文件，自动去重，无分组联系人手动分配分组，存入独立二进制文件
2. 脱敏导出CSV：手机号中间4位替换*，导出分组、黑名单状态
3. 脱敏导出TXT文本：格式化展示脱敏联系人，原始源文件数据不改动

### 6. 数据可视化统计分析
1. 主通讯录统计：总人数、各分组人数占比字符柱状图、正常/黑名单人数分布图表
2. VCF导入数据独立统计：分组分布、黑白名单可视化字符图表
3. 短信24小时发送时段分布统计，自动识别短信高峰时段

### 7. 加密备份与数据恢复
1. 一键加密备份：对VCF数据源使用XOR异或加密（密钥0x68），按时间戳生成备份文件
2. 备份恢复：输入备份文件名解密覆盖现有VCF数据，恢复前强制输入yes确认，防止误操作

### 8. 系统操作日志
1. 全流程日志记录：添加/删除/修改/拉黑/排序/群发短信/系统自动清理等操作全部记录
2. 日志展示：打印操作人、关联手机号、操作行为

### 9. 内置通用工具函数
1. 手机号合法性校验（严格限制11位纯数字）
2. 手机号脱敏函数（中间4位替换为*）
3. 时间戳工具：获取当前时间、同年同月同日对比、时间戳转为“X月X日”字符串
4. 联系人重复检测、缓冲区清空、XOR二进制加密
5. 程序初始化：首次运行自动创建全部存储文件，无需手动新建
